# TDL WinterSchool Final Project

This project tests this website - http://www.automationpractice.pl/index.php

## How to install
Run command:
```
npm install
```

## How to run tests
Run command:
```
npm run wdio
```

## How to run tests using tags
Run command:
```
npm run tag -- "@tagname"
```

## How to view report
First run the tests and then run command:
```
npm run allure
```
